# 📊 Diagrams Folder

Save your report diagrams here. Each person should contribute:

- **Person 1:** Linked List diagram (add/remove operations)
- **Person 2:** Queue diagram (FIFO) + Stack diagram (LIFO)
- **Person 3:** Min Heap diagram (tree + array representation)
- **Person 4:** Graph diagram (weighted edges) + Dijkstra trace table
- **Person 5:** BST diagram (tree structure) + HashMap diagram (buckets + chaining)

Accepted formats: `.png`, `.jpg`, `.drawio`, `.pdf`

You can use tools like:
- [draw.io](https://app.diagrams.net/) — free, works in browser
- PowerPoint / Google Slides — for simple diagrams
- Hand-drawn + photo — if neat and clear
